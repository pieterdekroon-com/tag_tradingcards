# TAG Trading Cards

A client-side trading card generator built with React and TypeScript. Design custom digital trading cards with live preview, custom themes, and image cropping.

## Features

- **Live card preview** — side-by-side editor with real-time card rendering
- **Custom themes** — gradient-based themes with rarity tiers (Common, Rare, Epic, Legendary)
- **Image upload & crop** — drag-and-drop upload with 4:3 aspect ratio cropping
- **Specialty tags** — add multiple specialties to each card
- **Card flip** — click to flip between front and back
- **JSON export** — export card data for use elsewhere
- **Fully client-side** — no backend, no data leaves your browser

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- Vite + React 19 (TypeScript)
- CSS Modules
- react-easy-crop
- Google Fonts (Syne)
